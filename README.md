# 🍽️ Restoran Sitesi - Frontend

Bu proje, modern bir restoran web sitesinin kullanıcı arayüzünü (UI) temsil eder. Kullanıcıların menüyü inceleyebileceği, rezervasyon yapabileceği ve şubeler hakkında bilgi alabileceği şık ve kullanıcı dostu bir tasarımdır.

## 🚀 Özellikler

- **Dinamik Slider:** Ana sayfada Swiper.js ile güçlendirilmiş etkileyici görsel geçişler.
- **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu arayüz.
- **Modern CSS:** CSS Değişkenleri (:root) kullanılarak yönetilebilir renk ve boyut paleti.
- **Sayfalar:** Anasayfa, Rezervasyon, SSS ve Şubelerimiz.

## 🛠️ Kullanılan Teknolojiler

- **HTML5 & CSS3** (Flexbox ve Grid sistemleri)
- **JavaScript (ES6+)**
- **Swiper.js** (Görsel kaydırıcı için)
